1. Routemaster - Optimal Path Guider

Developed a Python application to find the shortest path between nodes in a graph using A* and Dijkstra's algorithms. Implemented both algorithms to compare their efficiency and performance. Represented graphs using adjacency lists and matrices. Designed heuristic functions for A* to optimize pathfinding. Conducted performance analysis on various graph types and sizes. Created a command-line interface for user input and path visualization. This project enhanced my skills in graph theory, algorithm design, and Python programming.

2. Battleship Game Using IPC (Inter-Process Communication)

Developed a Battleship game utilizing Inter-Process Communication (IPC) with shared memory to enable real-time multiplayer gameplay. The game allows multiple processes to interact by sharing a common memory space, ensuring efficient data exchange between players. Synchronization mechanisms such as semaphores and message queues were implemented to prevent data inconsistency and manage concurrent access. The system ensures smooth gameplay by handling player moves, updating the game board, and verifying win conditions in real time.

3. Drone System For Hydro Powerplant

Developed a drone-based monitoring system for hydro powerplants, integrating AI-powered fault detection and real-time surveillance. The system utilizes UAVs equipped with high-resolution cameras and thermal sensors to capture aerial footage and identify structural anomalies such as cracks or leaks using OpenCV and TensorFlow. It employs LoRa/WiFi for remote data transmission and cloud platforms like AWS/ThingSpeak for real-time monitoring and analysis. The drones also assess water flow and sedimentation levels, enhancing predictive maintenance. By automating inspections and reducing manual intervention in hazardous areas, the system improves safety, operational efficiency, and cost-effectiveness in hydro powerplant maintenance.

Utilized React to create a responsive and dynamic user interface, ensuring an intuitive user experience. Leveraged MongoDB for efficient data storage and retrieval, enabling seamless management of large datasets. Integrated RESTful APIs for communication between the frontend and backend. Ensured data security and user authentication using JWT. This project improved my skills in full-stack development, React, and NoSQL databases.

5. Maze Game using socket programming

6. Human Activity Recognition

Developed a machine learning model using Long Short-Term Memory (LSTM) networks to predict human activities based on sensor data. The model processes time-series data to classify activities such as walking, running, sitting, and standing. It leverages deep learning techniques to capture temporal dependencies, improving prediction accuracy.

7. Spam Mail Detection

Developed a machine learning model to classify emails as spam or not using an ensemble approach of six different models. The system preprocesses email text by tokenizing, removing stop words, and vectorizing the content using TF-IDF. Each email is then analyzed by multiple classifiers, including logistic regression, random forest, XGBoost, and support vector machines. The predictions from these models are combined using a voting mechanism to enhance accuracy and minimize false positives. The final system effectively filters out spam emails while maintaining high precision for genuine messages.

The implementation optimizes memory usage and provides a scalable solution for handling large word datasets with minimal search time complexity.
